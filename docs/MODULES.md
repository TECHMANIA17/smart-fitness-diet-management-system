# Module Explanation

## 1. User Module
- User registration and login using JWT authentication.
- User profile stores age, gender, height, weight, activity level, fitness level, and goal.
- Automatic BMI, BMI category, ideal weight, calorie, and water calculations.
- Personalized dashboard with diet plan, workout plan, and recent progress.
- Progress tracker to store daily fitness records.
- History page to view previous entries in sorted order.

## 2. Admin Module
- Admin login with secure authentication.
- Dashboard with summary counts for users, plans, tips, and progress records.
- Manage users with list and delete operation.
- Manage diet plans using add, edit, and delete.
- Manage workout plans using add, edit, and delete.
- Manage tips using add, edit, and delete.
- View user progress records for monitoring.

## 3. Business Value
- Gyms can track members and suggest plans.
- Trainers can monitor performance and improve recommendations.
- Diet consultants can maintain structured meal plans.
- Health centers can provide consistent wellness advice through one system.
