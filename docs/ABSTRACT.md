# Project Abstract

Smart Fitness and Diet Management System is a dynamic web application developed for gyms, fitness trainers, diet consultants, and health centers. The system helps users evaluate their BMI, identify ideal weight range, estimate daily calorie needs, monitor hydration, and receive personalized diet and workout suggestions based on their fitness goal and activity level. It also includes progress tracking so users can record weight, calories, water intake, steps, workouts, and notes over time.

The application contains separate User and Admin modules. The User module supports registration, login, profile management, automated health calculations, a personalized dashboard, and history tracking. The Admin module supports secure login, user management, and CRUD operations for diet plans, workout plans, and fitness tips. MongoDB Atlas is used as the cloud database, Node.js and Express.js handle backend business logic, and Netlify deployment support is included for practical web hosting. This project is suitable for a college mini project because it demonstrates business functionality, dynamic data handling, CRUD operations, secure authentication, and cloud deployment.
